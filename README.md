# pi-Maestro-flow

Maestro workflow tools as [Pi](https://github.com/earendil-works/pi) extensions.

## Packages

| Package | Description |
|---------|-------------|
| [pi-teammate](./pi-teammate/) | Teammate dispatch with P0 three-axis decoupling (name × reply_to × lifecycle) |
| [pi-maestro-agent](./pi-maestro-agent/) | Explore, delegate, and MOA tools built on pi-teammate |

## Getting Started

```bash
# Install both packages
pi install ./pi-teammate
pi install ./pi-maestro-agent

# Or from npm
pi install npm:@pi-maestro/teammate
pi install npm:@pi-maestro/agent
```

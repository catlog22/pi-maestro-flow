# Tone Guidelines for Rebuttal Writing

Reference document for maintaining professional academic tone in review responses.

## Core Principles

1. **Always thank**: Every response begins with gratitude
2. **Stay respectful**: Even when the reviewer is wrong
3. **Be objective**: Use fact-based, evidence-supported language
4. **Be constructive**: Provide alternatives when disagreeing
5. **Acknowledge weakness**: Honest about limitations

## Expression Reference

### Gratitude Expressions

| Context | Expression |
|---------|------------|
| General thanks | "We thank the reviewer for this comment/suggestion" |
| Valuable insight | "We thank the reviewer for this valuable/insightful feedback" |
| Important suggestion | "We thank the reviewer for this excellent/important suggestion" |
| Error caught | "We thank the reviewer for catching this/pointing this out" |

### Agreement Expressions

| Context | Expression |
|---------|------------|
| Full agreement | "We agree that..." / "We concur with the reviewer that..." |
| Partial agreement | "We agree that [part], however..." |
| Acknowledge issue | "We acknowledge this limitation/concern" |
| Accept suggestion | "We have adopted this suggestion" |

### Disagreement Expressions (Polite)

| Context | Expression |
|---------|------------|
| Polite disagreement | "We respectfully note that..." |
| Provide rationale | "However, we chose [method] because..." |
| Explain constraints | "While [suggestion] would be valuable, it is not feasible due to..." |
| Offer alternative | "Instead, we have [alternative], which..." |

### Clarification Expressions

| Context | Expression |
|---------|------------|
| Point to existing | "We would like to clarify that..." |
| Admit unclear writing | "We apologize for the confusion. What we meant is..." |
| Provide location | "This is discussed in Section X (page Y)" |
| Improve wording | "To make this clearer, we have..." |

## Prohibited Expressions

- "The reviewer is wrong"
- "The reviewer misunderstood"
- "The reviewer failed to notice"
- "This is obvious"
- "Clearly..."
- "It is well-known that..."
- "Everyone knows..."
- "Done." / "Fixed." (too terse)

## Tone Checklist

### Must Include
- [ ] Every response starts with thanks
- [ ] Use "We" not "I"
- [ ] Specific location references (Section, page, Table, Figure)
- [ ] Concrete improvement actions stated
- [ ] Professional and respectful throughout

### Must Avoid
- [ ] No "reviewer is wrong" phrasing
- [ ] No arrogant vocabulary
- [ ] No defensive or aggressive tone
- [ ] No vague promises without specifics
- [ ] No skipped comments
